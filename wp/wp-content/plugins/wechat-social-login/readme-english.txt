== Description ==
Wechat Social Login plugin is professionally developed by Xunhuweb and is a WordPress (WooCommerce and Ultimate Member compatible) plugin that allows your visitors to comment, login and register with many Social Media applications like for example WeChat, QQ, Weibo, LinkedIn, Github.

= Features =
* Extension model, according to the requirements to install extensions
* login and register with many Social Media applications like for example WeChat, QQ, Weibo, LinkedIn, Github.
* Support login short code, PHP code
* A new regist and login page replace the wordpress native page
* Compatible with Buddypress
* Compatible with Open Social
* Support account binding settlement tied (Charge extension)
* Support mobile regist and login (Charge extension)
* Integration of the Ultimate Member (Charge extension)
* Integration of the WooCommerce (Charge extension)
* WeChat login Plus , do not need WeChat open platform developers certification(Charge extension)
